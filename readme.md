All the code in this repository comes from the freely available paper:

http://www.jaeckel.org/LetsBeRational.pdf

The HH3 approximation has not been implemented, but the initial "zero"
approximation is already very good for non-industrial purposes. 

Jaeckel's implementation is used, for example, by MATLAB:

https://www.mathworks.com/help/finance/blsimpv.html
